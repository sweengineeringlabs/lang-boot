# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please:

1. **Do NOT** open a public issue
2. Email security@engineeringlab.dev with details
3. Include a proof of concept if possible
4. Allow up to 90 days for a fix before public disclosure

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Security Best Practices

When using Pyboot:

- Keep dependencies updated
- Use the security module for authentication
- Validate all user input with the validation module
- Use secure defaults
